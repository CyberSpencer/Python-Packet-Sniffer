<h2>Description</h2>
This project demonstrates a Python-based packet sniffer tool that captures and analyzes network packets. The tool allows users to inspect packets in real-time and filter traffic based on various criteria. The primary objective of this project is to provide a hands-on understanding of network analysis and cybersecurity.

<h2>Languages and Utilities Used</h2>

- <b>Scapy</b>
